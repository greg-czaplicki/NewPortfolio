Grocery List App

A Django based app that allows the user to add and categorize items on their grocery list. Items can have weights and/or quantity associated with them. Items can be clicked/tapped to mark as 'completed'. Basic CRUD operations used to manage the database

CT Brewery Trail App

A single page application featuring a list of breweries inthe state of Connecticut. Built using Javascript, Knockout.js, jQuery, HTML/CSS and Yelp API. The app allows you to filter the brewery list down as you type. In return, it displays the marker on the map with Yelp information window.
